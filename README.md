## Prerequisites

For building the project Python3 and [Cheetah3](https://pypi.org/project/Cheetah3/) package required.

## Build with CMake

```
mkdir cmake_build
cd cmake_build
cmake -DCMAKE_BUILD_TYPE=Release ..
cmake --build .
```

See also build scripts in `build_scripts` directory.
